## 🔧 技术架构

### 后端架构
- **框架**: Go 1.21+ + Gin Web框架
- **数据库**: GORM + SQLite/PostgreSQL
- **缓存**: Redis
- **日志**: 结构化日志系统
- **中间件**: JWT认证、性能监控、安全防护
- **依赖注入**: 统一容器管理

### 前端架构
- **框架**: Vue 3 + Composition API
- **UI库**: Element Plus
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **构建工具**: Vite
- **工具布局**: 统一的ToolsPageLayout组件

### 核心功能特性
- ✅ 用户认证与权限管理
- ✅ 实时WebSocket通知系统
- ✅ RESTful API设计
- ✅ 响应式设计，支持移动端
- ✅ 统一错误处理
- ✅ 性能监控与日志记录
- ✅ 安全防护机制
